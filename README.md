# Legion Live

Legion Live is a real-time streaming, creator battle, and video platform built for competitive live content, audience interaction, and creator monetization.

Inspired by gladiatorial combat and Roman battle culture, Legion Live combines live streaming, PK battles, short-form video, and digital gifting into a single platform.
